# NUC_cityWall - 虚幻引擎FPS项目

## 📌 项目说明
本项目基于 **B站 UP主 SRiver星露** 发布的《虚幻引擎5 FPS游戏开发教程》开源内容制作，用于团队学习与开发实践。

教程原视频地址：[BV1aw9jBeEHr](https://www.bilibili.com/video/BV1aw9jBeEHr)

## 📜 开源声明
本项目的基础内容由 SRiver星露 公开分享，**完全免费、开源**，遵循原作者的开源意愿，可自由学习、修改与共享。
- 本仓库仅用于团队内部学习与项目备份，不做商用或二次分发
- 所有基础教程内容版权归原作者所有

## 🛠️ 环境与版本
- 引擎版本：Unreal Engine 5.x
- 开发语言：C++ / Blueprints
- 项目用途：FPS游戏开发学习、团队协作练习

## 📂 仓库内容
- 项目主文件：`NUC_cityWall.uproject`
- 游戏资源：角色模型、材质贴图、音效资源
- 代码模块：基础移动系统、武器系统、AI逻辑、射击手感实现

## 🚀 如何运行
1.  克隆本仓库到本地
2.  使用对应版本的 Unreal Engine 打开 `NUC_cityWall.uproject`
3.  等待资源加载完成即可启动编辑器
